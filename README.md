## Course Structure

- Each day of training has its own folder (for example, `Day1`, `Day2`, etc.).
- Every day folder contains practical tasks and exercises.
- Tasks are designed to build skills step by step, from basics to more advanced topics.

## Goal

The goal of this course is to practice Python programming while applying it to networking-related problems and scenarios.

## How to Use This Repository

1. Open the folder for the current day.
2. Complete the tasks in order.
3. Test and improve your solutions.
4. Move to the next day folder when done.


## Highlights by Day

### Day1
- **ip_validator.py**: Validates IPv4 addresses entered by the user.

### Day2
- **Command-Line Subnet Calculator.py**: Calculates network and broadcast addresses and usable hosts for a given IPv4/CIDR.
- **test.py**: Simple number input validation.

### Day3
- **task.py**: Parses firewall logs, analyzes actions, and generates CSV, JSON, and threat reports.
- **output.csv**, **output.json**, **threats.txt**: Outputs generated by the log analysis script.

### day5
- **test.py**: Placeholder for exercises.

### day6
- **task1.py**, **task2.py**: Network automation and scripting tasks.

### day7
- **ssh_with_pkey.py**: Demonstrates SSH connection using a private key with Paramiko.

### day8
- **nmap_scanner.py**: Scans a network for live hosts and saves results in JSON, YAML, and XML formats.
- **converter.py**: Loads and displays scan results from different formats.
- **task2.py**: Automated configuration backup for network devices using Netmiko.
- **task2.md**: Task description and sample outputs for network automation.
- **backups/**, **raw_ping_reports/**: Output folders for backups and scan results.

### day9
- **task.py**: Nmap scan with email notification of results (using yagmail).
- **scan_results.json**: Output from the scan.

### Project
- **project.py**: SSH audit tool for network devices, checks for Telnet, HTTP, and SNMP settings, and generates a dated audit report.
- **Audit_Report_YYYY-MM-DD.txt**: Example audit report generated by the script.

This repository is a comprehensive hands-on Python and Networking course, with each day building on practical skills for automation, security, and network management. See each day's folder for more details and code examples.
